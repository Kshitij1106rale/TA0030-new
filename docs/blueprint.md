# **App Name**: VeriHire AI

## Core Features:

- Candidate Registration: Allow candidates to register and create profiles.
- HR Registration: Enable HR personnel to register and manage hiring processes.
- Document Upload & OCR: Allow candidates to upload documents; simulate OCR to extract key data.
- AI-Driven Risk Assessment: Use a simulated AI tool to compare data and calculate fraud and trust scores.
- Agentic Workflow Automation Simulation: Simulate AI agents for document extraction, data comparison, fraud detection, and trust score generation.
- Interactive Dashboards: Visualize candidate data and risk assessments on interactive dashboards using Chart.js.
- Admin Action Simulation: Admin Approves or rejects candidates, this data gets saved to the Cloud database

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) for a professional and trustworthy feel.
- Background color: Light gray (#F0F0F0) for a clean, modern look.
- Accent color: Electric Violet (#8F00FF) to highlight interactive elements and CTAs.
- Body font: 'Inter', a sans-serif for clean and modern readability.
- Headline font: 'Space Grotesk', a sans-serif for futuristic headings.
- Use a set of professional, consistent icons related to data verification, security, and AI, with subtle animations to enhance user experience.
- Implement a fully responsive design with a clean, modular layout, optimized for mobile, tablet, and desktop screens.
- Use smooth, subtle transitions and animations for interactive elements and data updates.
